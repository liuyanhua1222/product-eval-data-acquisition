# credentials.md — 数据平台登录凭证

> ⚠️ **安全说明**：本文件存储各数据平台的登录凭证。请勿将此文件分享给任何人。
> 凭证以 base64 编码存放（防误读取，非真正加密，但比明文强）。
> 如需更高安全性，建议使用系统密钥环（Keychain）替代。

## 填写说明

格式：
```
[平台名称]
username: <用户名或手机号>
password: <密码或验证码>
notes: <备注，如账号类型、有效期等>
last_updated: <最后更新时间 YYYY-MM-DD>
```
